## Description

ThumbYT is a lightweight Flask web application that automatically generates high-quality thumbnail images from any video—whether from a YouTube URL or a local file upload. Using yt-dlp for downloading and ffmpeg for frame extraction, it offers a clean interface to customize the number of thumbnails and time range, with real-time progress tracking and batch download as a ZIP file. Perfect for content creators, developers, or anyone needing quick, automated thumbnails.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/pudszttiot/ThumbYT.git
   cd ThumbYT
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
   Note: Ensure ffmpeg is installed on your system and available in your PATH (required for frame extraction).

## Usage
1. Run the tool using the below command:
   ```bash
   python app.py
   ```
2. Open your browser and go to http://127.0.0.1:5000/.

3. Enter a YouTube URL in the input field or click the upload area to select a local video file (supports MP4, MOV, AVI, MKV, WEBM, FLV, WMV).

4. Set the number of thumbnails you want (between 1 and 20).

5. (Optional) Specify a start and end time (format MM:SS or HH:MM:SS) to extract thumbnails from a specific segment.

6. Click the Generate button. A progress bar will show download and extraction status.

7. Once processing completes, you'll be redirected to the results page where you can preview each thumbnail and download them all as a single ZIP file.

## Features

<ins>**YouTube URL Support:**</ins>

   - Extract thumbnails from any public YouTube video by simply pasting the link.

<ins>**Local File Upload:**</ins>

   - Upload your own video files (MP4, MOV, AVI, MKV, WEBM, FLV, WMV) and generate thumbnails locally.

<ins>**Customizable Thumbnail Count:**</ins>

   - Choose exactly how many thumbnails to generate—from 1 to 20—to suit your needs.

<ins>**Time Range Selection:**</ins>

   - Define a custom start and end time to focus thumbnail generation on a specific part of the video.

<ins>**Real-time Progress Tracking:**</ins>

   - Watch the download and extraction progress live with a dynamic progress bar.

<ins>**Batch Download:**</ins>

   - Download all generated thumbnails in a single ZIP file for easy storage and use.
